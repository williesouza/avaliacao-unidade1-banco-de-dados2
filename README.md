## Descrição do Projeto

Este repositório contém a atividade da Unidade 1, que implementa os seguintes requisitos com base no schema de banco de dados fornecido:

1.  **10+ Tabelas:** O schema base fornecido já contém 13 tabelas.
2.  **Mapeamento Objeto-Relacional (SQLAlchemy):** O arquivo `modelos_sqlalchemy.py` contém todas as classes Python mapeadas para as tabelas do banco de dados.
3.  **4 Stored Procedures:** O script `sql/01_stored_procedures.sql` contém as 4 procedures solicitadas.
4.  **3 Triggers:** O script `sql/02_triggers.sql` contém os 3 triggers solicitados e suas funções de suporte.

## Estrutura do Repositório

* `/modelos_sqlalchemy.py`: Código Python com o mapeamento ORM.
* `/requirements.txt`: Dependências Python para executar o projeto.
* `/sql/00_schema_base.sql`: Script original para criação da estrutura do banco de dados.
* `/sql/01_stored_procedures.sql`: Script com as 4 Stored Procedures.
* `/sql/02_triggers.sql`: Script com os 3 Triggers.
